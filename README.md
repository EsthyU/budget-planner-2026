# 💰 Annual Budget Planner 2026

A luxurious, fully automated financial dashboard — works as a **Progressive Web App** on iPhone and Android.

## Features

- 📱 **Save to iPhone Home Screen** — works like a native app
- 💱 **Multi-currency**: RON, EUR, USD, GBP with live conversion
- ⚡ **Fully automated**: Edit any number → everything recalculates
- 🔴 **Debt tracker**: Click to mark payments paid → totals update live
- 💼 **Income streams**: Track 5 income sources across 10 months
- ✅ **Action plan**: 23 tasks with status tracking
- 💎 **Savings tracker**: Editable targets with growth chart
- 🔒 **100% private**: All data stays on your device

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
npm install --save-dev gh-pages
npm run deploy
```

Then in GitHub → Settings → Pages → Source: `gh-pages` branch.

Your site: `https://YOUR-USERNAME.github.io/budget-planner-2026/`

### Save to iPhone Home Screen

1. Open the site in **Safari** on your iPhone
2. Tap the **Share** button (square with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Name it "Budget 2026" → tap **Add**
5. It now appears as an app icon — opens fullscreen with no browser bars!
